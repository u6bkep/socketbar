Socketbar — filters

Hide listeners that aren't typically web-facing. The popup has a Show all toggle that temporarily bypasses every filter on this page.

Processes
One per line or space-separated. Case-insensitive substring match against the process name.
If non-empty, only listeners from these processes are shown. Overrides the denylist.
Ports
Space- or line-separated. Ranges allowed: 22 25 3000-3100.
If non-empty, only these ports are shown. Overrides the denylist and the noisy-process rule below.
Heuristics
Listeners from these processes on ports ≥ threshold are hidden. Port allowlist overrides this.
